# Ahmed Mobile Screen IDs

استخدم هذه الأرقام عند طلب تعديل شاشة محددة.

## الشاشات الرئيسية

| الرقم | الشاشة | الملاحظات |
|---|---|---|
| S-001 | ثروتي | الشاشة الرئيسية الحالية للتطبيق |
| S-002 | احصائيات | لوحة الاحصائيات العامة |
| S-003 | تقارير | مركز التقارير |
| S-004 | استثماراتي | قائمة منصات الاستثمار |
| S-005 | مزيد | الاختصارات والاعدادات |

## شاشات الاستثمار

| الرقم | الشاشة | الملاحظات |
|---|---|---|
| S-100 | تعميد | شاشة فرص تعميد والفلاتر الحديثة |
| S-110 | حسابات المستثمرين | شاشة مستقلة لاختيار المستثمر |
| S-120 | موني مون | شاشة استثمارات موني مون |
| S-130 | دينار | غير مفعلة/قريبا |
| S-140 | ترميز | غير مفعلة/قريبا |

## شاشات تعميد الفرعية

| الرقم | الشاشة | الملاحظات |
|---|---|---|
| S-101 | فلتر تعميد | المستثمر/التصنيف/الحالة داخل شاشة تعميد |
| S-102 | تفاصيل فرصة تعميد | تفاصيل وسجل دفعات فرصة تعميد |
| S-103 | سداد تعميد | نافذة لصق رسالة استلام تعميد واعتمادها |
| S-104 | اختيار فلتر تعميد | نافذة اختيار المستثمر أو التصنيف أو الحالة |

## شاشات حسابات المستثمرين الفرعية

| الرقم | الشاشة | الملاحظات |
|---|---|---|
| S-111 | شاشة المستثمر الرئيسية | احصائيات المستثمر وروابط الشاشات الفرعية |
| S-112 | إدارة حركات أرصدة المستثمر | اضافة رصيد، سحب، استيراد، تعديل، حذف |
| S-113 | الحركات المالية لكل مستثمر | كل الاستلامات والايداعات والسحوبات |
| S-114 | تفصيل فرص المستثمر | تفصيل كل فرصة ومبلغ المستثمر والمستلم والمتبقي |

## قاعدة التطوير

- عند إنشاء شاشة جديدة بديلة وأفضل يجب حذف الشاشة القديمة أو فصلها نهائيا عن أي مسار تنقل.
- لا يتم الرجوع للشاشات القديمة بعد اعتماد شاشة جديدة.
- أي شاشة جديدة يجب أن تحمل رقم شاشة واضح في العنوان أو في وصف البطاقة.
